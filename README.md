## Pre-merge review

Pull requests can request an automated read-only merge risk review by adding this comment:

```text
/pre-merge-review
```

The workflow runs in GitHub Actions, analyzes the pull request diff, and posts or updates a single review comment from `github-actions[bot]`. It has read-only access to repository contents and cannot push commits to the pull request branch.
